  1. Lệnh cơ bản

  mkdir ten_thu_muc

  Ví dụ tạo thư mục tên website:
  mkdir website

  2. Nếu tên có dấu cách

  Phải dùng dấu nháy " " hoặc dấu \:
  mkdir "Cua hang laptop"

  3. Một vài lệnh hữu ích đi kèm

  ┌─────────────────────────────────────────┬───────────────────────────────┐
  │                Mục đích                 │             Lệnh              │
  ├─────────────────────────────────────────┼───────────────────────────────┤
  │ Xem mình đang ở thư mục nào             │ pwd                           │
  ├─────────────────────────────────────────┼───────────────────────────────┤
  │ Xem các file/folder trong thư mục hiện  │ ls                            │
  │ tại                                     │                               │
  ├─────────────────────────────────────────┼───────────────────────────────┤
  │ Đi vào một thư mục                      │ cd ten_thu_muc                │
  ├─────────────────────────────────────────┼───────────────────────────────┤
  │ Đi ra thư mục cha (lùi 1 cấp)           │ cd ..                         │
  ├─────────────────────────────────────────┼───────────────────────────────┤
  │ Tạo nhiều cấp thư mục cùng lúc          │ mkdir -p                      │
  │                                         │ website/images/laptop         │
  └─────────────────────────────────────────┴───────────────────────────────┘

  4. Ví dụ trọn vẹn (làm thử)

  cd ~/Documents          # đi tới thư mục Documents
  mkdir cua-hang-laptop   # tạo thư mục mới
  cd cua-hang-laptop      # đi vào thư mục vừa tạo
  pwd                     # kiểm tra: đang đứng ở đâu

  5. Cách mở terminal trên Mac

  - Nhấn Cmd + dấu cách → gõ Terminal → Enter
  - Hoặc mở Launchpad → Other → Terminal
